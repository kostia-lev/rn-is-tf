# rn-is-tf

Is testflight check for iOS. Returns false on Android

## Installation


```sh
npm install rn-is-tf
cd ios. && pod install
```


## Usage


```js
import { isTestFlight } from 'rn-is-tf';

// ...

const isTestflight = isTestFlight();
```
